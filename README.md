# SimFace

Simple watchface for pebble

![simface](https://raw.githubusercontent.com/kandeshvari/simface/master/resources/simface.png)

# Build, Check, Install
        
        pebble build
        pebble install --emulator diorite
        pebble install --phone 10.0.1.11
